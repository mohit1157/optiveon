# Optiveon

Optiveon is an advanced algorithmic trading research and market analysis platform.

## Project Structure

This repository contains two main components:

1.  **Landing Page** (Root Entry): The marketing website for Optiveon.
    -   Static HTML/CSS/JS located in the root directory.
    -   Entry point: `index.html`.

2.  **Web Application** (`optiveon-web`): The SaaS platform for users.
    -   Built with Next.js, PostgreSQL, and Prisma.
    -   Located in the `optiveon-web/` directory.

## Getting Started

To work on the web application:

```bash
cd optiveon-web
npm install
npm run dev
```

See `optiveon-web/README.md` for more details on the application setup.
